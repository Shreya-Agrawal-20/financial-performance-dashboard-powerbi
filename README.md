An interactive Power BI dashboard analyzing sales, profit, and performance metrics across countries, products, and customer segments using the Financial Sample dataset.

## 🎯 Project Overview
This dashboard provides a complete view of business performance — from high-level KPIs to deep-dive analysis of segments, discount bands, and product profitability. Built to practice real-world Data Analyst skills including data modeling, DAX, and interactive dashboard design.

## 🔑 Key Features
- **KPI Cards:** Total Sales, Total Profit, Total COGS, Profit Margin %
- **Sales Trend Analysis:** Monthly sales trend line chart
- **Country-wise Performance:** Sales comparison across 5 countries
- **Product Analysis:** Sales distribution by product (donut chart)
- **Segment Analysis:** Sales performance by customer segment
- **Discount Band Impact:** How discounts affect profitability
- **Country vs Product Matrix:** Cross-analysis table
- **Detailed Data Table:** Full transaction-level data
- **Interactive Slicers:** Filter by Country, Segment, and Product

## 🛠️ Tools & Skills Used
- **Power BI Desktop** – Dashboard design & visualization
- **Power Query** – Data cleaning and transformation
- **DAX** – Custom measures (e.g., Profit Margin %)
- **Data Modeling** – Relationships between Financials and Date tables

## 📈 Key Insight
The "None" discount band shows the lowest profit contribution, while "Low" discount band drives the highest profit — suggesting a potential opportunity to optimize discount strategy.
